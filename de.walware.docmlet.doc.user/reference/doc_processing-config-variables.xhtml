Configuration - Variables

The following special variables are available in document processing steps:

Name Description
source_file_path Returns the workspace relative path of the source document file
resource_path Returns the workspace relative path of the input file of the current step, if no argument is specified
out_file_path Returns the workspace relative path of the output file of the current step
out_file_ext Returns the file name extension of the output file, defined by the format, of the current step

Beside 'resource_path', also the other common resource variables ('resource_name', 'resource_loc', 'resource_enc', 'container_name', 'container_path', 'container_loc', 'project_name', 'project_path', 'project_loc') refer to the input file of the current step, if no argument is specified:

Expression Description
${container_loc} Returns absolute file system path of the container of the input file
${resource_enc} Returns the encoding of the input file

Combining Variables

The variables 'source_file_path' and 'out_file_path' resolves, like other variables ending with '_path', to the workspace relative path. It is possible to wrap this variables inside other variables like 'resource_loc', 'resource_name', 'container_loc' or 'project_loc':

Expression Description
${resource_loc:${source_file_path}} Returns the absolute file system path of the source document file
${resource_name:${source_file_path}} Returns the file name of the source document file
${resource_loc:${out_file_path}} Returns the absolute file system path of the output file of the current step
${container_loc:${out_file_path}} Returns the absolute file system path of the output container of the current step
